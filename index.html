Position no CSS

Position: static

A propriedade position: static é o valor padrão do posicionamento de elementos em CSS. Quando um elemento está com position: static, ele é posicionado de acordo com o fluxo normal da página, ou seja, sua posição é determinada pela ordem em que aparece no código HTML, sem ser afetado pelas propriedades de deslocamento como top, right, bottom ou left. Esse tipo de posicionamento não cria um novo contexto de empilhamento nem permite sobreposição de elementos. Em resumo, o static é utilizado quando não se deseja alterar o comportamento natural de posicionamento de um elemento dentro do layout da página.

Position: relative

A propriedade position: relative em CSS permite posicionar um elemento em relação à sua posição original no fluxo da página. Quando aplicada, o elemento mantém seu espaço reservado no layout, mas pode ser deslocado usando as propriedades `top`, `right`, `bottom` e `left`. Por exemplo, `top: 10px` move o elemento 10 pixels para baixo de onde ele normalmente estaria. Além disso, um elemento com `position: relative` pode servir de referência para elementos com `position: absolute` dentro dele. Em resumo, esse posicionamento é útil para fazer pequenos ajustes visuais sem alterar o fluxo natural da página.

Position: absolute

A propriedade position: absolute em CSS é usada para remover um elemento do fluxo normal da página e posicioná-lo em relação ao seu primeiro ancestral com position diferente de static (ou em relação ao corpo do documento, se não houver nenhum). Isso significa que o elemento não ocupa mais espaço no layout, podendo ser colocado exatamente onde o desenvolvedor quiser, usando as propriedades top, right, bottom e left. Esse tipo de posicionamento é muito útil para criar layouts complexos, sobreposições, caixas flutuantes e elementos que precisam ficar fixos em uma área específica da tela ou de um contêiner.

Position: fixed

A propriedade position: fixed em CSS fixa um elemento em uma posição específica da janela de visualização (viewport), independentemente da rolagem da página. Isso significa que, mesmo quando o usuário rola o conteúdo, o elemento permanece sempre visível no mesmo lugar da tela. Ele é posicionado usando as propriedades top, right, bottom e left, e é removido do fluxo normal do documento, não ocupando espaço no layout. Esse tipo de posicionamento é comumente usado para criar cabeçalhos fixos, barras de navegação, botões de “voltar ao topo” e outros elementos que devem permanecer acessíveis durante toda a navegação.
